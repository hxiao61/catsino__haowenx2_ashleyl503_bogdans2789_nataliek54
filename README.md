## Project Descriptions
Catsino is a gambling website where users can "Play to Win" with our internal currency, Tuna. 
The platform features four main games: Blackjack, Poker, Slots, and Roulette

Players can log in and track their stats and use their winnings to buy collectible cat icons

This website uses "Deck of Cards API" for card games, "The Cat API" for cats, and "Evil Insult API" to taunt players for losing
